# 📊 Dashboard Xbox Game Pass — Subscriptions Sales

> Dashboard de vendas desenvolvido no Excel como desafio do Bootcamp TOTVS — Fundamentos de Engenharia de Dados e Machine Learning.

---

## 📌 Sobre o Projeto

Dashboard interativo com análise de vendas de assinaturas do Xbox Game Pass, desenvolvido a partir de uma base de dados real fornecida pelo desafio.

O dashboard responde as seguintes perguntas de negócio:

- Qual o faturamento total de vendas separado por Auto Renovação?
- Qual o total de vendas de assinaturas do EA Play Season Pass?
- Qual o total de vendas de assinaturas do Minecraft Season Pass?

---

## 📈 Visualizações

- **Cards de KPI** — Total de vendas EA Play e Minecraft Season Pass em destaque
- **Gráfico de barras** — Total de assinaturas Xbox Game Pass separado por Auto Renovação (Yes/No)
- **Filtro interativo** — Segmentação por tipo de assinatura (Annual, Monthly, Quarterly)

---

## 🛠️ Tecnologias Utilizadas

| Ferramenta | Uso |

| `Excel` | Desenvolvimento do dashboard completo |
